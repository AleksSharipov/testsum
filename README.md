# Реализовать приложение (SPA), которое умеет показывать следующие страницы:
- / главная
- /login — страница ввода логина и пароля
- /registration — форма из нескольких полей, недоступна авторизованным
пользователям
- /profile — страница с приветствием + имя пользователя + кнопка выхода
(logout), недоступная без авторизации

## Функционал
- Для авторизации ввести "username" из "db.json" и любой пароль. Валидация для авторизации отсутствует.
- Для успешной регистрации нужно заполнить все поля с учетом валидации.
- Не авторизованному пользователю не доступна страница "/profile".

## Технологии
- react
- reduxjs/toolkit
- react-router-dom
- axios
